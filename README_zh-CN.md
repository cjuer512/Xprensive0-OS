# 关于本项目
**为什么叫这个名字**  
我六年级的时候有段时间自己造了很多英语词。起名的时候我偷懒了，直接拿的当时造的一个词用了，没什么含义。没错，我就是这么随意。  

**为什么不发表release**  
我更新是小而快，那版本号变得就太快了，而且github服务器不在中国境内，速度慢。  

**为什么编译不用cmake而是用命令行脚本**  
因为我不会用cmake  

**有没有考虑引入其它语言**  
没有，我觉得intel风格汇编和c语言就挺好，我不想引入c++和rust（因为一个复杂到不可控开销大，另一个编译太慢我电脑带不动）

# 编译
**怎么编译**  
在linux操作系统（我用的windows子系统wsl）下进行以下步骤：  
1. 安装nasm（汇编器），gcc（c语言编译器）  
2. 在终端中写“./build.sh”，生成虚拟硬盘，启动（时间太长了，我忘了生成的临时目录里有没有纯二进制而不是虚拟硬盘的）
